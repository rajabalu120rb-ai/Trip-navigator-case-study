# Trip Navigator App – UX Case Study
Overview:

Trip Navigator is a mobile travel planning app that simplifies booking, organizing, and managing trips in one seamless platform.

Problem:

Travelers struggle to manage flights, hotels, and itineraries across multiple apps.

Solution:

A unified mobile app that simplifies booking and itinerary tracking.

Tools Used:

Figma, Miro, FigJam

Role:

UX Research, Wireframing, UI Design

Outcome

Delivered a user-friendly travel solution that reduces planning complexity and improves booking efficiency.
